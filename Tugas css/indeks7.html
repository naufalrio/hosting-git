Kemnaker Luncurkan Program Desmigratif

Program Desmigratif dari Kemnaker

Liputan6.com, Jakarta Menteri Ketenagakerjaan (Menaker), M. Hanif Dhakiri, meluncurkan secara resmi Program Desa Migran Produktif (Desmigratif) di 120 desa pada Senin (11/9/2017). Desmigratif merupakan desa yang sebagian besar penduduknya bekerja di luar negeri, memahami sistem penempatan dan perlindungan tenaga kerja, baik di dalam maupun di luar negeri. "Peluncuran ini bertujuan untuk memantapkan kesamaan persepsi dalam pelaksanaan Program Desmigratif di 60 Kabupaten/Kota dan 120 desa terpilih," ujar Hanif
